Summary

The purpose of this project was to analyze different weather properties in relation to latitude. The dataset used in this project derive from a previous project where I was tasked with randomly generating over 500 latitudes, finding the nearest city, and then pulling weather data from the OpenWeatherAPI by latitude.

All analysis and graphing in this project was performed using Matplotlib, Pandas and Numpy. Elements of the weather studied in this project include maximum temperature, cloudiness, humidity, and wind speed. This site contains all the visualizations created, as well the dataset used.
